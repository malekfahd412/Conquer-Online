Task 11 — Ticket System Pro (Phase 4)

Objective

Build a professional Question Builder for Ticket Panels.

This replaces the current simple 5-question implementation.

This should become a complete Form Builder.

Everything must be configurable inside the Ticket Panel Designer.

No JSON editing.

No slash commands.

Everything from Discord.

====================================================

Question Builder

====================================================

Every panel can contain multiple forms.

Each form can contain up to 5 Discord modal fields.

Admins can create unlimited forms.
